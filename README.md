# Enumerables_Ruby

Building an Enumerable module with the next methods:

- #my_each
- #my_each_with_index
- #my_select
- #my_all?
- #my_any?
- #my_none?
- #my_count
- #my_map
- #my_inject

We also modified #my_map in order to accomplish the requirements to accept blocks and Procs
